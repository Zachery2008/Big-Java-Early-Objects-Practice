Comparing Objects

  • == tests for identity, equals for identical content
  • Rectangle box1 = new Rectangle(5, 10, 20, 30);
    Rectangle box2 = box1;
    Rectangle box3 = new Rectangle(5, 10, 20, 30);
  • box1 != box3,
    but box1.equals(box3)
  • box1 == box2
  • Caveat: equals must be defined for the class
