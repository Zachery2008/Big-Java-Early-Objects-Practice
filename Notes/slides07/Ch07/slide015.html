do Loops

  • Executes loop body at least once:
    do
       statement
while (condition);

  • Example: Validate input

    double value;
    do
    {
       System.out.print("Please enter a positive number: ");
       value = in.nextDouble();
    }
    while (value <= 0);
  • Alternative: 
    boolean done = false;
    while (!done)
    {
       System.out.print("Please enter a positive number: ");
       value = in.nextDouble();
       if (value > 0) done = true;
    }
