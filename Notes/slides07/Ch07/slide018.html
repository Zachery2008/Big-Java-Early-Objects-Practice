for Loops

  • for (initialization; condition; update)
       statement
  • Example: 
    for (int i = 1; i <= n; i++)
    {
       double interest = balance * rate / 100;
       balance = balance + interest;
    }
  • Equivalent to
    initialization;
    while (condition)
    { statement; update; }
  • Other examples: 
    for (years = n; years > 0; years--) . . .
    for (x = -10; x <= 10; x = x + 0.5) . . .
