The Generalized for Loop

  • Traverses all elements of a collection:
    double[] data = . . .;
    double sum = 0;
    for (double e : data) // You should read this loop as "for each e in data"
    {
       sum = sum + e;
    }
  • Traditional alternative:
    double[] data = . . .;
    double sum = 0;
    for (int i = 0; i < data.length; i++)
    {
       double e = data[i];
       sum = sum + e;
    }
