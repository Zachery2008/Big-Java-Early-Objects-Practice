Two-Dimensional Arrays

  • When constructing a two-dimensional array, you specify how many rows and columns you need:
    final int ROWS = 3;
    final int COLUMNS = 3;
    String[][] board = new String[ROWS][COLUMNS];
  • You access elements with an index pair a[i][j] 
    board[i][j] = "x";
