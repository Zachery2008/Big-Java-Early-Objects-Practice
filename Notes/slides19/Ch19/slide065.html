CompareTo Method

  • The implementation must define a total ordering relationship
    • Antisymmetric
      If a.compareTo(b) = 0, then b.compareTo(a) = 0
       
    • Reflexive
      a.compareTo(a) = 0
       
    • Transitive
      If a.compareTo(b) = 0 and b.compareTo(c) = 0, then a.compareTo(c) = 0
