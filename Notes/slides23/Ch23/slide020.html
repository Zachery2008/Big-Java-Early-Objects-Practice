Terminating Threads

  • The run method should check occasionally whether it has been interrupted
    • Use the interrupted method
    • An interrupted thread should release resources, clean up, and exit 
      public void run()
{
   for (int i = 1;
         i <= REPETITIONS && !Thread.interrupted(); i++)
   {
      Do work
   }
   Clean up
}
