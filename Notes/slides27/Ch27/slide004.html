A Simple JSF Program

  • Each JSF page has the following structure: 
    <html>
   <%@ taglib uri="http://java.sun.com/jsf/core" prefix="f" %>
   <%@ taglib uri="http://java.sun.com/jsf/html" prefix="h" %>
   <f:view>
      <head>
         <title>Page title</title>
      </head>
      <body>
         <h:form>
            Page contents
         </h:form>
      </body>
   </f:view>
</html>
